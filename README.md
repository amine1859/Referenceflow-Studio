# Referenceflow-Studio
A lightweight desktop reference board for designers, architects, and artists featuring always-on-top floating reference windows, AI-powered image search, drag &amp; drop, transparency controls, multi-monitor support, collections, image annotations, and a distraction-free workflow

## Features
- 🖼️ Floating always-on-top reference windows
- 🖥️ Multi-monitor support
- 🎯 Click-through mode
- 🔍 AI-powered image search
- 🌐 Search across multiple online sources
- 📂 Collections and project organization
- 🖱️ Drag & drop images directly into creative applications
- 🔎 Zoom, pan, and rotate images
- ✏️ Quick sketches and notes
- 🎨 Adjustable opacity and transparency
- 📌 Lock/unlock reference windows
- ⚡ Lightweight Electron desktop application
- 🚀 Fast startup and responsive performance
- 💾 Local project saving
- 🖼️ High-resolution image support
- 📐 Designed for architecture, graphic design, concept art, industrial design, and creative workflows
